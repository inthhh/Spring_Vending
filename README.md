# FullStack 개발 - 자판기 만들기

<strong>[ API 명세 ]</strong>
- /api/products (GET) : 전체 데이터 목록 조회
- /api/add (POST) : 데이터 추가
- /api/product?index= (GET) : 단일 데이터 조회
- /api/edit (POST) : 데이터 수정
- /api/remove (DELETE) : 데이터 삭제
